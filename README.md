## 这是一个简单的工具集合
### 集合里面包括 dom操作，ajax请求，日期函数，事件处理，动画相关函数


```markdown

git clone https://github.com/Jankid/utils
yarn install
yarn run dev
直接访问 http://localhost:8060 

```
